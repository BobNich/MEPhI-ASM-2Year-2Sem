Here are prerequiremenets:
    1. Инициализировать first_word_complete = FALSE
    2. Инициализировать r8 (first_word_length) = 0
    3. Инициализировать r9 (current_word_length) = 0

Алгоритм:
    1. Проверить, пробел ли:
        Если пробел -> 4.
        Если не пробел -> 1.1.

    1.1. Проверить, таб ли:
        Если таб -> 4.
        Если не таб -> 1.2.

    1.2. Проверить, '/n' ли:
        Если '/n' -> 4.
        Если не '/n' -> 1.3.

    1.3. Проверить, \0 ли:
        Если \0 -> 6.
        Если не \0 -> 2.

    2. Проверить, первое ли слово:
        Если первое слово -> 2.1.
        Если не первое слово -> 2.2.

    2.1. Посчитать длину первого слова:
        Если first_word_start:
            Сохранить указатель на первую букву первого слова
        Добавить к first_word_length 1 -> 3.

    2.2. Посчитать длину текущего слова:
        Если current_word_length:
            Сохранить указатель на первую букву текущего слова
        Добавить к current_word-length variable 1 -> 3.

    3. Перейти к следующей букве после обработки буквы из слова:
        Проверить, пройден ли цикл обработки буфера (Cycle counter = Current buffer length):
            Если выполнен -> 5.
            Если не выполнен -> (перейти к следующей букве) -> 1.

    4. Перейти к проверке следующей буквы после символов space/tab/'/n':
        Если first_word_start != NULL -> 5.
        Если current_word_start != NULL -> 5.
        Если '/n' -> (Добавить \n в output_buffer) -> (перейти к следующей букве) -> 1.
        Иначе -> 
            Проверить, пройден ли цикл обработки буфера (Cycle counter = Current buffer length):
                Если выполнен -> 
                    Проверить last_word_undone = TRUE:
                        Если да -> (offset = длину последнего слова (неоконченного)) -> 6.
                        Иначе -> 5.
                Если не выполнен -> (перейти к следующей букве) -> 1.

    5. Записать слово в output_buffer:
        Если первое слово:
            Записать в output_buffer начиная с сохраненной позиции первого слова и посчитанной его длины
            first_word_start = NULL -> (перейти к следующей букве) -> 1.
        Если не первое слово:
            Проверить, равны ли current_word_length и first_word_length:
                Если равны -> записать в output_buffer начиная с указателя на текущее слово и посчитанной его длины
            current_word_start = NULL -> (перейти к следующей букве) -> 1.
